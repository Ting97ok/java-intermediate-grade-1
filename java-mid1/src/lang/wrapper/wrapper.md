# 래퍼, Class 클래스

## 래퍼 클래스 - 기본형의 한계
### 1. 객체가 아님
- 메서드 제공 불가 
### 2. null 값을 가질 수 없음

## 래퍼 클래스 - 자바 래퍼 클래스
### 자바 래퍼 클래스 특징
- 불변 객체
- `equals`로 비교해야 한다.

### 래퍼 클래스 생성자말고 valueOf()를 권장하는 이유
- `valueOf()`는 캐시를 사용하여 성능 향상
- 캐시는 String Pool과 비슷한 개념
- -128 ~ 127까지 캐시를 사용한다. 

## 래퍼 클래스 - 오토 박싱
### 오토 박싱/언박싱
- 컴파일 단계에서 자동으로 기본형 -> 래퍼, 래퍼 -> 기본형으로 자동 변환
